# Project CIE — Iteration 1

A grayscale narrative prototype for **Confidence Is Engineered**.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Before publishing

1. Replace `axel@example.com` in `components/cie-experience.tsx`.
2. Review the project claims and exact metrics.
3. Replace or expand the Origin section with a portrait only in a later iteration.
4. Add a real contact endpoint when moving beyond the MVP.

## Scope

This version validates:

- narrative order;
- information hierarchy;
- readability and spacing;
- two Engineering Decision case studies;
- restrained reveal motion;
- responsive behavior;
- reduced-motion accessibility.

It intentionally excludes accent color, photography, advanced diagrams, GSAP, Three.js, scroll hijacking and decorative effects.
